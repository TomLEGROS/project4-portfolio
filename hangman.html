face People Involved : Tom

"This project was a personal project that I did right before entering the Gaming Tech school. I wanted to create a Hangman Game just because I thought that it was one of the best way to confirm my bases in Python language. Let's get to the point, a hangman game is a one player game where a word is choosen randomly, then you'll have to guess the word by typing a letter, if you found a letter, it'll appear on the blanks but if you don't, it'll draw one line, but be careful, when you fail too much, the lines will form a hangman and you'll loose !"

"It wasn't really hard, in fact, Python is a high-level programming language so it was way easier to force the user to put a letter and not a number for example. The more challenging part of this project was to display the word with the missing letters but at the end, it was enriching."

access_time Time restriction : No Deadline

As it was a personal project, I had no deadlines but to be honest, it was quick because Python made the thing easier.

check Objectives

  • arrow_downwardRandomised choice of a word

    The first objective was to create a dictionnay that contains a bunch of random words so that my code could choose one random word like every hangman game do.

  • arrow_downwardDetection on wether your letter is in or not

    Next step was to ask the user to enter a letter and to verify if this letter is present in the random word, in the case of yes, it'll obviously put the letter at the place that it belongs, if not you'll loose a 'life'.

  • arrow_downwardDetection when you managed to find all the letters

    And last but not least, the win condition, I had to check if all the blanks were filled and if the word was correctly written and spelled.
