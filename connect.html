face People Involved : Tom, Hugo

"This project in C language was harder than the first because we originally had 2 games to code. It was also for most of the people, the first time they were coding in C, and as it considered as a low-level language, it was basically harder and longer to assimilate the bases. The Connect 4 game is a 2 player versus game where you'll have to put tokens on a fixed-sized grid with the aim of aligning 4 of your tokens, vertically, horizontally or even diagonally. It is a strategical game where you'll hae to manage to block your opponents advance while slowly reaching your own virctory."

access_time Time restriction : 3 weeks

We originally had 3 weeks to ode 2 games but since it was the first time coding in C for most of people so the deadline wa short and our intervener put the last project as a bonus so we could completely focus on the Connect 4. 3 weeks is more than enough to entirely code the game and to avoid every errors that the user can create with wrong inputs and so on.
