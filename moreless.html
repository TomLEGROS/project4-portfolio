face People Involved : Tom, Hugo

"This project was our first project at the Gaming Tech school. This project's goals was to introduce all the students to algorithms and Python language... The more or less game is a single-player game where a random number is chosen between an interval, then your goal is to find that number with the computer saying if the random number is more or less higher than the number that you've written"

"We found this project pretty easy because we already practiced Python and algorithm and did personal projects but we think that it was good to strenghten our already existing bases."

access_time Time restriction : 1 Week

The time limit was really acceptable in proportion to the work we had to do.

check Objectives

  • arrow_downwardRandomised choice of a number betwwen an interval

    The first objective was to make the computer choose a random number between a fixed interval so the game could begin.

  • arrow_downwardMake the computer say if it's more or less

    Next step was to make the computer to the comparaison between the number to find and your written number so it could help you by saying if you have to write a higher or lower number.

  • arrow_downwardMake the win / loose condition

    To finish, we had to initiate a number of maximal attempts that will determine if the player lost in the case where he didn't found the number in a given amount of attempts, or if he won by finding the good number.
